## How to run the application

## Prerequisite
1. `locate .env inside the server directory`
2. `change the URI to anything you like to connect to your MongoDB`

### Frontend
1. `cd client`
2. `npm run start:frontend`
### Backend
1. `cd server`
2. `npm run start:backend`
